# Kora teh Explorer's Blog

Kora is a fictional persona for an Alien on Earth who is documenting their thoughts based on the latest news.    
This is intended to be a fun learning experiment for understanding agentic processes while using small LLMs (in this case, a tiny model running on a `Surface Pro 4`).
    
The blog can be accessed at [https://forzabarca88.github.io/kora-teh-explorer/](https://forzabarca88.github.io/kora-teh-explorer/)
